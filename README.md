## 一些废话

关于AI的使用，飘子逸其实非常熟悉，从军训做政工开始就熟练掌握了GPT提示词的编写，还经过了各种水课作业的千锤百炼，可以说是炉火纯青。

但是，说真的，在飘子逸在大一挣扎的这一年里，尽管耳朵里充斥着AI相关的词汇，也听过几回关于AI的讲座，但是对于AI基本上没有过系统的了解，
